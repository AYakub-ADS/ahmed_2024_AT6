This file has information about class UserInputValidator.

It has a method called validate_positive_integers which recieves a list of strings as a parameter. 
It iterates over the list and checks if each element is a integer or not.

If it is an integer it adds it to a new list.

After iterating over all elements it returns the new list with all the valid integers in it.

It also has a method validated_message that prints a message when the list is validted successfully.
